# Dubsar Store 🛍️

تطبيق متجر ملابس عراقي احترافي مبني بـ React

---

## ✨ المميزات

### 🛒 للعملاء
- **واجهة عربية كاملة RTL** - تجربة تصفح سلسة وطبيعية
- **عرض المنتجات** - صور، أسعار، أحجام وألوان
- **سلة التسوق** - إضافة/حذف/تعديل الكميات
- **نظام الطلبات** - بيانات التوصيل، اختيار المحافظة
- **توصيل لجميع محافظات العراق** - أجور توصيل مخصصة
- **تصميم ديناميكي** - متوافق مع جميع الأجهزة
- **تنبيهات فورية** - رسائل نجاح وأخطاء

### ⚙️ لوحة التحكم (Admin Panel)
- **إدارة الطلبات** - عرض وتحديث حالة الطلبات
- **إدارة المنتجات** - إضافة/تعديل/حذف المنتجات
- **رفع الصور** - ضغط تلقائي وتخزين الصور
- **إدارة الإعدادات** - أجور التوصيل حسب المحافظة
- **إشعارات الطلبات الجديدة** - تنبيهات فورية كل 6 ثوانٍ
- **حماية برمز دخول** - Admin Passcode

## 🚀 البدء

### المتطلبات
- Node.js 14+
- React 18+
- npm أو yarn

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/cars28028-hue/Dubsar_store.git

# الدخول للمجلد
cd Dubsar_store

# تثبيت المكتبات
npm install

# تشغيل التطبيق
npm start
```

---

## 📦 المكتبات المستخدمة

- **React** - مكتبة الواجهات
- **lucide-react** - أيقونات احترافية
- **Tailwind CSS** - تنسيق الأنماط (مدمج في الكود)
- **Cairo & Tajawal Fonts** - خطوط عربية جميلة
